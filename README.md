Scala-Projekt für das Kartenspiel **Fantastische Reiche** / **Fantasy Realms**.

## Usage

Dies ist ein normales sbt-Projekt.

- Kompilieren: `sbt compile`
- Testen: `sbt test`
- Coverage: `sbt clean coverage test coverageReport`
- Starten: `sbt run`
- REPL: `sbt console`
